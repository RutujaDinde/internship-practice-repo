# Task 1 - Git & GitHub

## Objective

Learn the basic Git and GitHub workflow.

## Tasks Completed

* Created a GitHub repository
* Cloned the repository to the local system
* Created a README.md file
* Checked repository status using `git status`
* Staged files using `git add`
* Committed changes using `git commit`
* Pushed changes using `git push`
* Verified the repository on GitHub

## Outcome

Successfully learned the basic Git workflow and version control using Git and GitHub.
